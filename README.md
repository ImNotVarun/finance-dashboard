# 💰 Finance Dashboard

A modern, responsive finance dashboard built with **React + Vite**, focused on clean UI, smooth interactions, and real-time-like analytics.

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS

---

## 📦 Requirements

Make sure you have:

* Node.js (>= 18)
* npm or yarn

---

## 🚀 Installation

```bash
# Clone the repo
git clone https://github.com/ImNotVarun/finance-dashboard.git

# Navigate into project
cd finance-dashboard

# Install dependencies
npm install

# Run development server
npm run dev

```
